HTML 的 style 属性

style 属性的作用：

提供了一种改变所有 HTML 元素的样式的通用方法。

样式是 HTML 4 引入的，它是一种新的首选的改变 HTML 元素样式的方式。通过 HTML 样式，能够通过使用 style 属性直接将样式添加到 HTML 元素，或者间接地在独立的样式表中（CSS 文件）进行定义。

您可以在我们的 CSS 教程中学习关于样式和 CSS 的所有知识。

在我们的 HTML 教程中，我们将使用 style 属性向您讲解 HTML 样式。

不赞成使用的标签和属性

在 HTML 4 中，有若干的标签和属性是被废弃的。被废弃（Deprecated）的意思是在未来版本的 HTML 和 XHTML 中将不支持这些标签和属性。

这里传达的信息很明确：请避免使用这些被废弃的标签和属性！

应该避免使用下面这些标签和属性：

标签描述
<center>定义居中的内容。
<font> 和 <basefont>定义 HTML 字体。
<s> 和 <strike>定义删除线文本
<u>定义下划线文本
属性描述
align定义文本的对齐方式
bgcolor定义背景颜色
color定义文本颜色

对于以上这些标签和属性：请使用样式代替！

HTML 样式实例 - 背景颜色

background-color 属性为元素定义了背景颜色：

<html>

<body style="background-color:yellow">
<h2 style="background-color:red">This is a heading</h2>
<p style="background-color:green">This is a paragraph.</p>
</body>

</html>

亲自试一试

style 属性淘汰了“旧的” bgcolor 属性。

HTML 样式实例 - 字体、颜色和尺寸

font-family、color 以及 font-size 属性分别定义元素中文本的字体系列、颜色和字体尺寸：

<html>

<body>
<h1 style="font-family:verdana">A heading</h1>
<p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
</body>

</html>

亲自试一试

style 属性淘汰了旧的 <font> 标签。

亲自试一试：设置字体的旧方法

HTML 样式实例 - 文本对齐

text-align 属性规定了元素中文本的水平对齐方式：

<html>

<body>
<h1 style="text-align:center">This is a heading</h1>
<p>The heading above is aligned to the center of this page.</p>
</body>

</html>

亲自试一试

style 属性淘汰了旧的 "align" 属性。

os.path.abspath(‘.’) 当前下的决定路径
os.path.split(路径) 拆分，后一部分时最后的文件或目录
os.path.splitext(路径) 拆分，后一部分时文件扩展名
os.path.join(旧路径，新的后部分路径) 连接形成新的路径 一般在创造目录时使用，配合os.mkdir os.rmdir，也可以配合os。walk形成完整路径
os.path.isfile(文件名或者目录名或path) 括号里的是佛为文件
os.path.isdir(文件名或者目录名或path) 是佛为目录
os.path.getsize(path) 获得path的字节大小

os.rename(文件名或路径) 修改文件名字
os.remove(文件名或路径) 删除文件
os.walk(路径）所有的 返回一系列元祖，每个内部由parents，dirnames，filenames 组成
os.listdir（路径） 返回一个list，只有一级的目录和文件名字 配合os.path.isdir() 和 os.path.isfile()使用
